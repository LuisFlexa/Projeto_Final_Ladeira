\doxysubsection{Detailed Description}
Abstract base class representing a JVM object. 

The \doxylink{class_object}{Object} class provides a common interface for all objects in the JVM runtime. It enforces the implementation of the \doxylink{class_object_a85a54313726bd08230589f2b1e6ceece}{object\+\_\+type()} method to return the specific type of the object.
