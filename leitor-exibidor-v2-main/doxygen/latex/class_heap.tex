\doxysubsection{Detailed Description}
Manages heap-\/allocated objects in a JVM 8-\/style memory model. 

The \doxylink{class_heap}{Heap} class is responsible for storing references to dynamically allocated objects. It follows a singleton design pattern to ensure a single heap instance. Objects can be pushed onto the heap, but explicit memory deallocation is not handled within this implementation. Garbage collection or explicit deletion is expected elsewhere.
