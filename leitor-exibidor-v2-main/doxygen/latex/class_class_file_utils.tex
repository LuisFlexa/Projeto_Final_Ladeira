\doxysubsection{Detailed Description}
A utility class that provides static methods for working with class files. 

This class contains various helper methods used to process and analyze class files, such as\+:
\begin{DoxyItemize}
\item Checking class file version
\item Comparing UTF-\/8 strings
\item Indentation handling for pretty printing
\item Converting stringstreams to C-\/style strings
\item Verifying the version of a given class file
\item Checking if the class matches the given filename and class name 
\end{DoxyItemize}
