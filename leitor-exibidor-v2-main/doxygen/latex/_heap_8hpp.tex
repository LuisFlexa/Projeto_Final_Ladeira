\doxysubsection{Detailed Description}
Defines the \doxylink{class_heap}{Heap} class for managing dynamic object allocation in a JVM 8 environment. 

This class implements a singleton heap where dynamically allocated objects (instances of the {\ttfamily \doxylink{class_object}{Object}} class) are stored. It follows a basic memory management model, keeping track of allocated objects in a vector.
