\doxysubsection{Detailed Description}
Represents the static portion of a Java class. 

In the JVM 8 context, a Java class is not only defined by its instance fields and methods, but also by its static fields and static initializers. The \doxylink{class_static_class}{Static\+Class} module encapsulates the class\textquotesingle{}s metadata (stored in a \doxylink{class_class_file}{Class\+File}) as well as a collection of its static fields. This allows the JVM to access and modify static variables during execution.
